Overview

프로젝트 개요

Reimagining a real brand's digital presence

실제 브랜드의 디지털 경험을
새롭게 설계하다

Prairie Cricket Farms is a family-owned Canadian brand producing sustainable cricket protein products — powders, snacks, and supplements — from their farm in Manitoba.

Prairie Cricket Farms는 캐나다 매니토바에 위치한 가족 운영 농장으로, 크리켓 파우더, 스낵, 서플리먼트 등 지속 가능한 크리켓 프로틴 제품을 생산하는 브랜드입니다.

The brand's mission — sustainable, nutrient-dense protein for everyday people — is compelling. But the digital experience didn't reflect that quality. I took on this project to build something that truly represents the brand.

지속 가능하고 영양이 풍부한 단백질을 일상에 가져온다는 브랜드 미션은 훌륭했지만, 디지털 경험은 그 가치를 충분히 담지 못했습니다. 이 프로젝트는 브랜드를 제대로 표현하는 웹 경험을 만들기 위해 시작됐습니다.

Using a premium wellness brand as a design reference, I created a new visual identity, design system, and complete e-commerce experience — all coded from the ground up.

프리미엄 웰니스 브랜드를 디자인 레퍼런스로 삼아 새로운 비주얼 아이덴티티와 디자인 시스템을 구축하고, 완전한 이커머스 경험을 처음부터 코딩했습니다.

작업 범위

  • Brand identity & visual design system

Key Interactions

주요 인터랙션

Details that elevate
the experience

경험을 한 단계 높이는
디테일

01
Cart Drawer
카트 드로어
Slide-in cart panel with item management, upsell block, promo progress bar, and checkout CTA. Body blur via CSS filter, ESC & overlay dismiss.
아이템 관리, 업셀 블록, 프로모 진행 바, 체크아웃 CTA가 있는 슬라이드인 카트 패널. CSS 필터 바디 블러, ESC 및 오버레이 닫기.
02
Account Drawer
계정 드로어
Multi-view auth panel (login / signup / forgot password) with Supabase integration and rolling text animation on buttons.
Supabase 연동 및 버튼 롤링 텍스트 애니메이션이 적용된 멀티뷰 인증 패널 (로그인/회원가입/비밀번호 재설정).
03
Sticky Cart Bar
스티키 카트 바
Fixed ATC bar that appears when the product hero scrolls out of view and hides before sticky parallax sections. RAF loop synced with Lenis.
제품 히어로가 뷰포트를 벗어날 때 나타나고 스티키 패럴랙스 섹션 직전에 숨겨지는 고정 ATC 바. Lenis와 동기화된 RAF 루프.
04
Mega Menu Navigation
메가 메뉴 네비게이션
Desktop mega menu with promo cards, mobile full-screen overlay. Header auto-hides on scroll down and restores on scroll up or hover.
프로모 카드가 있는 데스크탑 메가 메뉴, 모바일 전체 화면 오버레이. 스크롤 다운 시 자동 숨김, 스크롤 업 또는 호버 시 복원.
05
GSAP Scroll Animations
GSAP 스크롤 애니메이션
ScrollTrigger-powered reveal animations, sticky parallax community banner, and IntersectionObserver counter animations.
ScrollTrigger 기반 리빌 애니메이션, 스티키 패럴랙스 커뮤니티 배너, IntersectionObserver 카운터 애니메이션.
06
Rolling Text Buttons
롤링 텍스트 버튼
Hover animation on all secondary buttons — text slides up to reveal a duplicate, creating a smooth rolling effect with CSS transforms.
모든 세컨더리 버튼에 적용된 호버 애니메이션 — 텍스트가 위로 슬라이드되어 복제본이 보이는 CSS 트랜스폼 기반 롤링 이펙트.
07
Accessibility
웹 접근성
WCAG AAA contrast on all primary UI, focus-visible outlines, prefers-reduced-motion support, semantic HTML, and ARIA labels throughout.
주요 UI WCAG AAA 대비, focus-visible 아웃라인, prefers-reduced-motion 지원, 시맨틱 HTML, 전체 ARIA 레이블.
08
Image Optimization
이미지 최적화
Next.js Image with conditional optimization (dev: off, prod: WebP + cache). Retina-ready assets at 2× resolution, lazy loading throughout.
조건부 최적화(개발: 비활성, 프로덕션: WebP + 캐시)가 적용된 Next.js Image. 2× 해상도 레티나 에셋, 전체 지연 로딩.
